# Electron Release Guide

## Local run
```bash
npm install
npm start
```

## Build Windows package
```bash
npm run build:win
```

## Verification checklist
1. Mở app và xác nhận đủ 8 tab.
2. Chạy Daily/Backtest/Monitor từ UI.
3. Xác minh log realtime và history cập nhật.
4. Xác minh Results preview/open hoạt động.
5. Xác minh Health API/DB/SMTP trả status hoặc Error Code.
6. Build Windows artifact thành công.
